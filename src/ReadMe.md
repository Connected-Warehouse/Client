# Comment utiliser le client ?

## Avoir d'installé :

- [ ] Avoir Python
- [ ] Avoir PIP (optionnel mais grandement recommandés)
- [ ] Avoir la librairie webview d'installé (scripts d'installations fournis pour linux/macOs et pour windows)

## Comment démarrer le programme ?

Une fois toutes les préconditions remplies, il vous suffira de lancer le main.py se trouvant dans gui.
